# Landen-Roupp-Projection-and-Analysis
Project Goals:
-Evaluate Roupp's traditional and underlying statistics 
-Compare his pitch mix and location quality to the early careers of Logan Webb and Aaron Nola 
-Build a three year projection using a Marcel style aging projection curve 
-Highlight strengths, weaknesses, and adjustments required to improve performance

Tools Used:
R
SQL
Baseball Savant
FanGraphs
Baseball Reference
Visual Studio Code
Gemini

Key Takeaways:
Roupp looks like a middle of the rotation starter with potential to grow if his ground ball profile sustains and if his command improves.
